## Comprehensive Internship Worklog Overview

A 13-week AWS internship at **AMAZON WEB SERVICES VIETNAM COMPANY LIMITED** (06/08/2025 - 12/24/2025) covering foundational AWS services (Weeks 1-5), core AWS concepts (Weeks 6-8), and Smart Office IoT project implementation with IoT Core, EventBridge, Lambda, and real-time data processing (Weeks 9-13).

---

### **Weeks 1-5: AWS Foundations**

**[Week 1: AWS Fundamentals & Account Setup](1.1-week1/)**
Secure AWS account setup with VPC networking, EC2 instances, and Lambda basics.

**[Week 2: Linux, Python, IAM & Databases](1.2-week2/)**
Linux administration, Python scripting, IAM identity management, and RDS fundamentals.

**[Week 3: Storage, Databases & APIs](1.3-week3/)**
S3 bucket management, DynamoDB NoSQL databases, and MQTT protocol for IoT.

**[Week 4: Serverless & Event-Driven Architecture](1.4-week4/)**
Lambda functions, S3 event triggers, API Gateway, and event-driven IoT patterns.

**[Week 5: Messaging Services & Project Preparation](1.5-week5/)**
SNS topics, EventBridge scheduling, AWS IoT Core overview, and project readiness.

---

### **Week 6-13: Smart Office IoT Project Implementation**

**[Week 6: IoT Core Environment Setup](1.6-week6/)**
AWS IoT Core initialization, device registration, certificate management, and MQTT connectivity.

**[Week 7: Security & Resilience Deep Dive](1.7-week7/)**
Enterprise security (IAM, encryption, WAF, GuardDuty), high availability, and disaster recovery.

**[Week 8: Performance, Cost Optimization & Midterm Exam](1.8-week8/)**
Storage optimization, RDS deployment strategies, cost management, and AWS Well-Architected Framework.

**[Week 9: MQTT Messaging & Data Simulation](1.9-week9/)**
MQTT topic hierarchy design and Python IoT device simulator with realistic sensor data streaming.

**[Week 10: IoT Rules & EventBridge Integration](1.10-week10/)**
AWS IoT Rules Engine with SQL filtering, EventBridge anomaly detection, and real-time data processing.

**[Week 11: Debugging, Code Refinement & Testing](1.11-week11/)**
System debugging, device scheduling implementation, code optimization, and production readiness.

**[Week 12: Linux & Shell Scripting Mastery](1.12-week12/)**
Linux commands, file manipulation, permissions, and shell scripting for infrastructure automation.

**[Week 13: Final Testing, Workshop & Presentation](1.13-week13/)**
End-to-end system testing, comprehensive workshop documentation, demo video, and successful project delivery.
